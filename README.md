# Student Result Dashboard

An interactive web application built with Python and Streamlit that analyzes student marks and visualizes performance.

## Features
- Calculate total marks
- Calculate percentage
- Assign grade
- Generate bar chart
- Generate pie chart
- Download result as CSV

## Tech Stack
Python
Streamlit
Pandas
Matplotlib
