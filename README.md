# Student Result Analysis Dashboard

## Live Demo
https://student-result-dashboard-n2gkrpglhhzyhfjeu5dhb5.streamlit.app/

## Description
This is an interactive web application built using Python and Streamlit to analyze student marks.

## Features
- Enter student marks
- Calculate total marks
- Calculate percentage
- Assign grade
- Generate bar chart
- Generate pie chart
- Download result as CSV

## Tech Stack
Python  
Streamlit  
Pandas  
Matplotlib

## How to Run

pip install -r requirements.txt
streamlit run app.py
